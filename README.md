# WPF 控件生成器 v2.0 (WPF Control Generator for LabVIEW) 🚀

**一个基于 AI 辅助开发的工具套件**，致力于为 LabVIEW 提供极其现代化、支持硬件加速与平滑交互的 WPF 前端控件体验。
它不仅是一套 DLL 控件库，更是一个“所见即所得”的零代码控件设计与一键导出引擎。

![WPF控件生成器 v2.0 界面预览](image.png)

## 🌟 核心理念
LabVIEW 拥有极为强大的硬件控制逻辑，但在原生 UI 上显得沉闷且过时。本工具基于 **C# WPF + LabVIEW .NET 容器互操作** 构建了一个完美的桥梁：
- **底层引擎**：LabVIEW 负责核心逻辑与通讯。
- **视觉前端**：WPF 负责圆角、阴影、渐变、悬浮高亮以及平滑阻尼动画。
- **全栈闭环**：无需 XAML 基础，一键可视化配置并生成供 LabVIEW 调用的专属 DLL。

## ✨ 已支持的核心“八大金刚”控件 (v2.0)
本项目目前已内置并在底层打通 LabVIEW 事件回调机制的八个高频控件：
1. `TextInput` - 文本输入框 (支持自适应高度)
2. `NumericDisplay` - 数值输出框 (高清晰度显示)
3. `Slider` - 阻尼滑动杆 (丝滑流畅)
4. `Button` - 流光动画按钮 (科技感十足)
5. `ComboBox` - 现代下拉列表
6. `LED` - 极简指示灯 (支持自定义开启/关闭颜色)
7. `ToggleSwitch` - 拟态开关
8. `ProgressBar` - 渐变进度条

## 📂 项目工程结构
```text
├── WPF控件生成器.exe    # [核心] 可视化设计器(Release正式版)
├── ControlDesigner/    # 可视化工具源码 (WPF)
├── ExportTemplate/     # 导出引擎模板工程库 (已优化，免 VS 即可编译)
├── Themes/             # 预设主题存储
├── UI/                 # 存放导出的 DLL 及详细说明文档
└── README.md
```

## 🚀 v2.0 重大更新说明 (2026-03)
1. **智能导出引擎**：用户电脑**无需安装 Visual Studio** 即可正常导出 DLL！系统会自动检索内置 .NET 框架进行编译。
2. **重命名与品牌化**：正式更名为“**WPF 控件生成器**”，并配有全新的 **高清 Neumorphic 科技感图标**。
3. **功能增强**：所有控件现在都支持通过属性控制标签（Label）的显示/隐藏（`SetLabelVisible`）。
4. **极致稳定性**：加固了颜色解析逻辑，具有“自愈”能力，杜绝了手动输入颜色导致程序崩溃的历史问题。
5. **版本固化**：程序集版本正式提升至 **2.0.0.0**。

## 🛠️ 环境支持
- **运行环境**：Windows 10 / 11 (内置 .NET 4.8)。
- **LabVIEW 兼容性**：支持 LabVIEW 2018 及以上版本 (32位与64位通用，示例 VI 采用 2020 编写)。
