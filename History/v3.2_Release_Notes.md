# LabVIEW WPF UI Generator - v3.2 Release Notes
*发布日期：2026-04*

## 🚀 从“局部重构”到“全场景覆盖”的终极进化

本次更新是继 v3.0 “交互革命”后的又一里程碑。如果说 v3.0 解决了“怎么动”的问题，那么 v3.2 则彻底解决了**“怎么摆”**与**“怎么看”**的问题。我们正式补全了页面级布局的最后一块拼图，并开放了全量字体生态。

### 1. 旗舰新构件：顶边导航栏 (TopbarNavPanel)
为了满足“传统仪表盘”与“现代管理后台”的双重需求，v3.2 引入了与侧边栏地位对等的横向导航方案。
* **水平布局生态**：补全了 Sidebar 以外的另一种主流导航范式，支持在 LabVIEW 容器顶部实现“流体拟态”导航。
* **动态项管理**：支持通过 API (`AddItem`/`ClearItems`) 实时增删导航标签，并自动计算间距。
* **深度视觉反馈**：
  * **选中态沉降**：点击项会从“凸起”平滑切换至“内凹”沉降态。
  * **底部高亮条 (Accent Indicator)**：底部配有可自定义颜色的呼吸指示条，强化当前页面感知。
* **三位一体 Logo 区**：左侧预留 Logo 展位，支持 **文字、Emoji 图标、外部图片** 的自由组合，支持像素级偏移微调。

### 2. 极简新成员：拟态图标按钮 (IconButton)
针对工具栏、设置面板等紧凑场景，我们推出了全新的 `IconButton`。
* **紧凑美学**：相比常规 Button，图标按钮将“图形”与“标签”进行了更深度的视觉聚合。
* **混合媒介支持**：支持 `UseImage` 模式切换。你可以选择使用轻量级的 **IconText (如 Emoji/字符图标)** 以获得极速渲染，也可以引用 **PNG/SVG 路径图片**。
* **一致性交互**：完美复刻了侧边栏的“阻尼触发”手感，当按钮被点击时，会有明显的物理按压回弹效果。

### 3. 核心突破：全量字体支持与样式穿透 (Typography Pro)
这是 v3.2 最受期待的底层优化，彻底终结了 LabVIEW 调用 .NET 控件时“字体难调”的尴尬。
* **全系统字体检索**：生成器内置了可视化字体选择器，能够自动识别 Windows 系统中安装的所有字体（如微软雅黑、Inter、HarmonyOS Sans 等）。
* **样式深度穿透**：
  * **全局覆盖**：字体设置不再仅局限于按钮表面，而是能够穿透至 `TreeView` 的每一级子节点、`DataGrid` 的单元格数据以及 `ComboBox` 的下拉项中。
  * **多国语言优化**：通过对渲染引擎的调整，v3.2 能够更完美地呈现越南文、韩文等特殊字符集。

## ✨ 已支持的全量核心控件矩阵 (v3.2)
本项目目前已内置并在底层完全打通 LabVIEW 事件与双通道色彩重绘回调机制的 **16 款** 高频工业控件：

1. `TextInput` - 文本输入框 (支持自适应与只读)
2. `NumericDisplay` - 数值输出框 (带独立单位格式渲染)
3. `Slider` - 阻尼滑动杆 (丝滑流体色彩)
4. `Button` - 流光动画按钮 (科技感触发)
5. `ComboBox` - 现代下拉列表 (高度可定制)
6. `LED` - 高级极简指示灯 (独立支持外围圆角及变色)
7. `ToggleSwitch` - 拟态物理感开关
8. `ProgressBar` - 动态渐变进度条
9. `ChartDisplay` - 折线图/多通道波形图 (带悬浮数值观测卡盘)
10. `PieDisplay` - 精致动态饼图 (中空光影渲染)
11. `GaugeDisplay` - 环形仪表盘 (动态极值重绘)
12. `DataGridDisplay` - 现代数据表格 (极简斑马线)
13. `TreeDisplay` - 树形展示面板 (动态复选与层级响应)
14. `SidebarNav` - 拟态侧边导航栏 (支持 UI 调度与 Logo 定制)
15. **`TopbarNav` - 拟态顶边导航栏 (✨ v3.2 NEW - 水平流体导航与指示器)**
16. **`IconButton` - 拟态图标按钮 (✨ v3.2 NEW - 支持 Emoji 与图片)**
