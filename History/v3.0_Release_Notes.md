# LabVIEW WPF UI Generator - v3.0 Release Notes
*发布日期：2026-04*

## 🍱 从“控件堆砌”到“系统交互”的跨代跃迁

v3.0 标志着本项目正式进入“页面级布局”时代。本次更新不仅带来了一个重量级的侧边导航容器，更在底层彻底解决了 LabVIEW 与 .NET 事件绑定时长期存在的兼容性痼疾。

### 1. 旗舰新构件：侧边栏导航面板 (SidebarNavPanel)
为了支撑多页面、高集成度的工业上位机界面，我们引入了全新的侧边栏构件。
- **拟态流体交互**：完美复刻拟态视觉，支持选中态与默认态的平滑切换，并配有毫秒级的展开/收缩阻尼动画。
- **全动态 Logo 展位**：支持通过 API 实时修改左上角的 Logo 文字、图标（Emoji）或外部引用图片，并允许通过 `SetLogoMargin` 进行像素级偏置调节。
- **左侧停靠优化**：专门针对 LabVIEW 容器进行了布局修正，确保侧边栏在收缩时向左侧边缘停靠，而非在容器中居中，从而腾出右侧最大的工作区。

### 2. 重大底层重构：LabVIEW 事件 100% 识别
针对以往用户反馈在 LabVIEW 中“找不到事件节点”的问题，v3.0 进行了毁灭性的重构：
- **定制化事件委托**：弃用了 .NET 通用的泛型 `EventHandler<T>`，转而使用了遵循 `[ComVisible(true)]` 规范的**专用自定义委托**。
- **零成本注册**：现在，在 LabVIEW 的事件结构或 .NET 事件注册节点中，您可以像使用原生 LabVIEW 控件一样，直接看到 `ItemSelected` 和 `StateChanged` 等命名精确的事件，无需任何手动转换。

### 3. API 与文档矩阵优化
- **语义化重命名**：统一了所有开放接口的命名规范，确保 `style.txt` 中的描述与 LabVIEW 下拉列表中的 API 完全一致。
- **交互逻辑反转**：根据社区反馈，将侧边栏默认状态调整为“弹出”效果，且选中态从“凹陷”修改为视觉更清爽的“扁平激活态”，极大提升了长时间使用的视觉舒适度。

## ✨ 已支持的全量核心控件矩阵 (v3.0)
本项目现已具备深厚实力提供打通 LabVIEW 事件与双通道色彩回调体系的 **14 款绝版高清交互工业构件**：

1. `TextInput` - 文本输入框 (支持自适应与只读)
2. `NumericDisplay` - 数值输出框 (带独立单位格式渲染)
3. `Slider` - 阻尼滑动杆 (丝滑流体色彩)
4. `Button` - 流光动画按钮 (科技感触发)
5. `ComboBox` - 现代下拉列表 (高度可定制)
6. `LED` - 高级极简指示灯 (开启全圆角设置)
7. `ToggleSwitch` - 拟态物理感感应断路器
8. `ProgressBar` - 动态渐变进度条
9. `ChartDisplay` - 多通道波形折线图 (带战术层数据卡盘)
10. `PieDisplay` - 精致动态中空饼图
11. `GaugeDisplay` - 环形仪表盘 (动态极值重绘)
12. `DataGridDisplay` - 现代极客数据表格
13. `TreeDisplay` - 树形展示面板 (节点无限推演与多状态捕捉)
14. **`SidebarNav` - 拟态侧边导航栏 (v3.0 重磅加入，支持 UI 页面调度与 Logo 定制)**
