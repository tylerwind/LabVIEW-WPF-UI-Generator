# LabVIEW WPF UI Generator - v2.5 Release Notes
*发布日期：2026-03*

## 🚀 核心架构升级 (Core Architecture Upgrades)
本次版本迎来史诗级图表引擎重构，正式从“单一波形渲染”跨入“现代化前端大屏看板”的标准。

### 1. 多通道无极扩展引擎 (Multi-Series Engine)
- **图表底层重写**：打破原有限制，`Chart` 与 `Pie` 控件现已全面升级原生多通道数据阵列（Multi-Series Arrays）。支持从 LabVIEW 直接将包含任意组曲线/饼图片段的数据连线“瞬查”到 C# WPF。
- **配置面板收折“折跃”方案**：消除了“无限罗列”的属性面板堆叠地狱。现在在控件设计器中，仅需在下拉菜单中切换通道 ID（如 Curve A、Curve B），配置下方的参数便会如同魔术般瞬间更换绑定的对象参数。

### 2. 旗舰级图表渲染体系 (Flagship Rendering System)
- **丝滑贝塞尔引擎 (Smooth Bezier Integration)**：`Chart` 折线图抛弃了工业时代的粗糙物理锯齿，支持实时启动贝塞尔级平滑算法。曲线下的阴影全部升级为半透明过渡色（Gradient Fills）。
- **光影环形饼图 (Donut DropShadow Pie)**：彻底告别纯平扁平大饼。全新 `Pie` 组件引入 Donut 中心镂空和实时 DropShadow 光源追踪算法阴影，高级感拉满。

### 3. “全局侦察战术级”数据浮台 (Floating Data Inspection Panel)
- **自带参数聚合看板**：图表的左上角被植入了带有精巧透明毛玻璃视效的半脱离属性卡台 (`ShowSeriesCards`)。
- **活数据追踪**：浮台将紧抓每一个曲线通道的名称、数值，饼图还能自动分摊和计算动态百分比率。
- **免切回实时调色 (Runtime Color Picker)**：用户或下位机使用者，只需要在图表的战术卡台上点击代表曲线的“小圆点”，便可无缝拉起系统级高级原生配色盘，即改即生效。

### 4. 彻底解决响应式崩盘的绝对钳制算法 (Absolute Clamping Responsive Algorithm)
- 历史久远的排版遗留问题宣告终结。在以前，关闭卡片常常导致饼图形单影只地偏在一个角落。
- **底层钳制引流**：引入了全新的 `UpdateCanvasSize` 算法引擎，将所有的波形与饼图严丝合缝地约束在弹性 Grid 包裹容器内。当悬浮观测看板消失的一瞬间，图表内核将重新演算空间尺度，0 延迟、0 虚位地物理回弹至屏幕绝对物理中轴线。
