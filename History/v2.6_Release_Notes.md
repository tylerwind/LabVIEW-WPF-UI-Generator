# LabVIEW WPF UI Generator - v2.6 Release Notes
*发布日期：2026-04*

## 🚀 国际化与底层架构极化升级

本次版本聚焦于上位机在极端部署环境下的系统兼容与多语言字符串传递时的痛点，不仅根除了一直以来困扰开发者的跨国区乱码问题，更在控件维度与打包策略上取得了跨越式的突破。

### 1. 全链路 UTF-8 数据穿透引擎 (UTF-8 Multi-language Bypass Engine)
长期以来 LabVIEW .NET 容器传递字符串由于 ANSI 编码强转在越南语、俄语等小语种环境下常会发生截断乱码现象。在此，我们重构了 13 款控件的双向文本通讯栈。
- **UTF-8 输入闭环**：大幅增设诸如 `SetLabelTextUTF8`, `WriteUTF8`, `AddSeriesUTF8`。LabVIEW 仅需向组件端口传输标准的 UTF-8 `byte[]` 字节数组波形，底层即刻自动完成无损拦截反反编译处理呈现，终结了传值的乱码折磨。
- **UTF-8 输出触发机制**：重大重构文本相关触发体系（包括 `Tree`, `ComboBox`, `TextInput`）。当控件被点击或文本变动引发拉取向 LabVIEW 的回调时，它在向外吐出时除了包含原参数，新增 `xxxUTF8` 数据针脚。LabVIEW 侧仅需勾选系统原生内置模块“字节流转字符串 (强指 UTF-8) ”即可还原任何多维小语种。

### 2. 阵列第十三人 - 多维动态树形展板 (TreeDisplay)
在原有的经典十二将阵列基础上，引入了具有高维度多层结构承压能力的高阶拟态工业构件—— `TreePanel`。由于高度工业化的设计，具备了以下核弹化特性：
- **无极阻尼展开**：多层级动态节点附带次世代流体平滑开启响应体验。
- **深网级事件捕捉反馈**：除节点双指连环触发防误触高级钩子 `NodeDoubleClicked`，更是打通底层实现了针对勾选的事件响应 `NodeChecked` 以及全尺寸热更新 `UpdateNodeText` / `UpdateNodeIcon`。
- **动态卡片化多列快捷菜单 ContextMenu**：告别 Windows 98 复古老旧菜单边框，自解构原生 `ContextMenu` 并成功内嵌扁平高级软阴影、高倍率呼吸 Padding 间距数值卡风格右键弹窗，并配备 `NodeMenuClicked` 实现无死角流向回调。
- **批量数据流收口**：调用全新方法 `GetCheckedNodes` 瞬间在后厨捞取数百个打勾网点，直插多向测试分发逻辑板芯栈。

### 3. MSBuild 双擎自适应降纬补偿架构 (Dual-Engine Framework Fallback)
由于各种目标厂房老旧工控机的系统版本参差不齐，以往部分系统强制脱离 4.0 包会引发生成 DLL 期间的爆栈（`_ErrorLog.txt`）。
- **现代高能起步**：引擎本体生成 C# 代码与主工程基础配置 `Template.csproj` 默认全面推向对标主流环境的 **.NET 4.8 框架**（可利用最新 VS2022 流水线）。
- **微秒级降级兜底**：一旦探测到宿主系统贫瘠不存在构建依赖抛出编译爆栈报错时，逻辑引擎毫秒级回压错误并在内存重写一切内部依赖强转匹配降极回溯至 **原生 .NET 4.0 版本**，发起第二次强制兜底编译并封口 DLL。无论全新电脑还是骨董机器，只管安心点一键生成！

## 🛠️ 其他工程与防呆机制更新
- **排雷级剔重机制**：对“生成合打包”的组件进行了逻辑切分。彻底清理并铲除了因循环导入说明文本而引起的 API 指南（`MyControlAll.style.txt`）里重复数次的段落冗余地狱。
- **参数解耦补齐**：全方位修复诸如三参数引脚强对两参数导致的编译器闪退警告。
- **自定义拓展补齐**：满足更多定制化使用，解放了 `LED` 硬件面板等控件的外源型自定义圆角属性接口调用。

## ✨ 已支持的全量核心控件矩阵 (v2.6)
本项目现已具备深厚实力提供打通 LabVIEW 事件与双通道色彩回调体系的 **13 款绝版高清交互工业构件**：

1. `TextInput` - 文本输入框 (支持自适应与只读)
2. `NumericDisplay` - 数值输出框 (带独立单位格式渲染)
3. `Slider` - 阻尼滑动杆 (丝滑流体色彩)
4. `Button` - 流光动画按钮 (科技感触发)
5. `ComboBox` - 现代下拉列表 (高度可定制下拉箭头色彩)
6. `LED` - 高级极简指示灯 (开启全圆角设置及高能多轨渲染技术)
7. `ToggleSwitch` - 拟态物理感感应断路器
8. `ProgressBar` - 动态渐变进度条
9. `ChartDisplay` - 多通道波形折线图 (带战术层数据卡盘侦测)
10. `PieDisplay` - 精致动态中空饼图
11. `GaugeDisplay` - 环形仪表盘 (动态极值重绘)
12. `DataGridDisplay` - 现代极客数据表格 (无缝斑马纹排线)
13. `TreeDisplay` - 树形展示面板 (节点无限推演与多状态捕捉能力首显)
