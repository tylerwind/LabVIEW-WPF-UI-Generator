## 🔙 破局者：回顾 v2.0 时代的“降维打击”

就在上个月，**v2.0 版本** 彻底引爆了上位机圈子，它解决了困扰 LabVIEW 开发者多年的“环境门槛”与“组件匮乏”两大痛点：

### 1. 智能导出引擎（告别 Visual Studio！）
在 v1.0 时，导出 DLL 还需要你的电脑里躺着一个庞大的 VS 环境。而在 v2.0 中，我重构了底层的导出逻辑，生成器会**自动调用你系统内置的 .NET 编译环境**，几秒钟内就能原地生成 DLL（甚至支持合集导出）。
**这是对“环境洁癖”工程师的终极救赎。**

### 2. “八大金刚”工控组件库成型
v2.0 把工控日常场景中最核心的控件全部“拟态化”了，补全了 LabVIEW 原生面板的短板：
- **LED 指示灯**：支持自定义开启/关闭颜色，比原生控件更高级。
- **拟态开关 (ToggleSwitch)**：拥有真实的触感反馈。
- **渐变进度条 (ProgressBar)**：动态视觉加持，数值展示更直观。
- **文本/数值输入框**：支持自适应高度和高清渲染。
- **阻尼滑动杆 (Slider)**：新增物理阻尼感。
- **流光动画按钮 (Button)**：自带呼吸动效交互。
- **各类图表 (Chart/Pie) 与 仪表 (Gauge)**：全面支持。

### 3. Neumorphic（拟态）科技感全新界面
整个工具箱的图标与 UI 彻底焕新，配合 `SetLabelVisible`（一键隐藏/显示标签文字）的 API 控制，让你做出的控制面板看上去就像是价值数万的商业套件。

## 🚀 史诗进化：v2.5 极致重构，彻底治愈你的强迫症！

如果说 v2.0 解决了“从无到有”的问题，那么 **v2.5 就是在细节体验上，做到了令人惊叹的“从有到极”**。

### 👑 战力一：告别反人类堆叠！Chart 与 Pie 引擎“升维”解耦
在旧版本中，如果给图表（Chart）加十条曲线，左侧就会无脑堆叠出十张长长的属性卡，导致界面排版崩溃直接越界。
在 v2.5 中，**所有序列的配置全部收纳进一张高度聚合的“折跃卡片”中**！只需在下拉框中优雅地选中指定 ID，即可独占式修改线条颜色、厚度和名称。无论是波形图还是饼图，界面清爽度直接飙升 500%！

### 👑 战力二：悬浮侦察兵！折线图专属“实时数据监测大屏”
图表不仅得有线，还得直观“看数”。v2.5 为 Chart 控件重磅引入了**“全局侦察兵”数值浮台 (Floating Inspection Card)**！
左上方会实时跟瞄弹出一个精致的半透明面板，动态同步展示每条曲线的**标签、专属色块以及最新抓取的实时数值**。只需打个勾开启 `ShowSeriesCards`，高级动态前端框架层才有的数据大屏质感，不用写一行额外代码就能实现。

### 👑 战力三：双轨制全域色彩引擎！极客体验拉满
“红蓝反转”彻底成为历史。v2.5 修复了 C# 与底层对 ARGB/RGB 内存排列理解的偏差。更为硬核的是，现在**所有控件的色彩接口**，全部升格为“双轨并行 API”：
- **Web前端极客流**：支持使用 `StartColor` (十六进制网页色号如 `#FF0000`)。
- **LabVIEW 原生党**：新增直接从底层调色板拉取的 `StartColorValue / ActiveColorValue`（标准 32 位 RGB 十进制整数，即 U32）。你的色彩调校从此如丝般顺滑。

### 👑 战力四：“黑匣子防摔”与零门槛超级全量 API 书
拿到 DLL 后面对一片空白不知道端子该连什么？v2.5 搭载了全新的自动全量 API 追踪引擎！你导出的 `style.txt` 末尾，现在会自动**挂载全部 12 款控件的所有读写属性、高级内部方法、色彩通道与触发事件的中文硬编码说明书**。零学习成本，开箱即接线！
另外，针对极端系统可能出现的 MSBuild 编译失败，新增了 `MyControlAll_ErrorLog.txt` 编译终端黑匣子日志机制，一秒定位错误，排错彻底告别玄学。

## 📋 致硬核开发者的：v2.5 深水区完整更新日志 (Changelog)

为了让大家看到此次重构的“吨位”与专业性，以下是本次大版本更新的技术清单，也是我们交出的一份底层重构答卷：

#### 🛠️ 渲染与内核修复 (Bug Fixes)
- **[Fix]** 图表 (Chart) 添加新 Series 时属性面板越界重叠崩溃的问题。
- **[Fix]** 饼图 (PieDisplay) 配置属性时偶发丢失焦点与数据绑定失效。
- **[Fix]** 底层内存色彩矩阵发生“红蓝通道翻转”的内部映射缺陷。
- **[Fix]** 仪表盘 (Gauge) 量程缩小时中心数值排版溢出与网格线畸变。
- **[Fix]** 开关 (ToggleSwitch) 导出后因后台事件隧道拦截导致的偶发“无法再次点击”的死锁 Bug。
- **[Fix]** 进度条 (ProgressBar)、滑动杆 (Slider) 导出 DLL 初始化引发自适应塌缩的问题。

#### 🌟 核心功能特性入库 (Features)
- **[Add]** **聚合图表卡集引擎**：摒弃无限堆叠，采用 `ComboBox` 路由动态绑定的方式来集中管理海量图表配置。
- **[Add]** **图表实时观测浮舱**：赋予 Chart 呈现左上角现代前端级的数据概览面板能力。
- **[Add]** **全面覆盖双通道色彩 API**：对底层所有控件组件均暴露 `*Color` (HEX) 和 `*ColorValue` (UInt32) 双格式供开发者连线自定。
- **[Add]** **原生全量说明挂载引擎**：大幅更新生成器内部硬编码，随文件附赠零漏点的开发者向全功能中文 API 说明。
- **[Add]** **ComboBox 独立倒三角宏**：下拉框加入 `ComboBoxArrowColor` 私有属性。
- **[Add]** **自动黑匣子捕捉器**：独立 MSBuild 控制台异步线程回调并打印原生 C# / MSBuild 编译全流程报错入 `MyControlAll_ErrorLog.txt`。
- **[Update]** 锁定组件构建器的全生命周期字符串编码（UTF-8 with BOM），掐断因系统区域/中文代码页造成的 CS1009 崩溃危机。
